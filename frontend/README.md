# Frontend (deploy simple)

This folder contains only the static frontend:
- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Configure backend URL
Edit `config.js` and set:
`window.__API_BASE__ = "http://YOUR_BACKEND:8000";`

## Open locally
You can open `index.html` directly, or serve it with any static server.
